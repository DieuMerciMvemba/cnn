## 📚 Références Scientifiques

### 1. Articles Fondamentaux

#### Parkhi, O.M., Vedaldi, A., & Zisserman, A. (2015)
**"Deep Face Recognition"**  
*Proceedings of the British Machine Vision Conference (BMVC)*  
- **Contribution** : Introduction de VGG-Face
- **Impact** : Architecture CNN standard pour reconnaissance faciale
- **Dataset** : 2.6M images, 2.6K personnes

#### Schroff, F., Kalenichenko, D., & Philbin, J. (2015)
**"FaceNet: A Unified Embedding for Face Recognition and Clustering"**  
*Proceedings of the IEEE Conference on Computer Vision and Pattern Recognition (CVPR)*  
- **Contribution** : Triplet loss et embedding facial
- **Performance** : 99.63% sur LFW
- **Application** : Google Photos

#### Taigman, Y., Yang, M., Ranzato, M., & Wolf, L. (2014)
**"DeepFace: Closing the Gap to Human-Level Performance on Face Verification"**  
*Proceedings of the IEEE Conference on Computer Vision and Pattern Recognition (CVPR)*  
- **Contribution** : 3D face alignment
- **Performance** : 97.35% sur LFW
- **Dataset** : 4.4M faces (Facebook)

---

### 2. Avancées Récentes

#### Deng, J., Guo, J., & Zafeiriou, S. (2019)
**"ArcFace: Additive Angular Margin Loss for Deep Face Recognition"**  
*IEEE Transactions on Pattern Analysis and Machine Intelligence*  
- **Innovation** : Angular margin loss
- **Performance** : 99.85% sur LFW
- **Application** : Large-scale recognition

#### Wang, M., & Deng, W. (2021)
**"Face Recognition at a Distance: A Survey"**  
*IEEE Transactions on Pattern Analysis and Machine Intelligence*  
- **Scope** : Reconnaissance faciale à distance
- **Défis** : Basse résolution, variations d'éclairage
- **Applications** : Surveillance, sécurité

#### Liu, W., et al. (2017)
**"Large Margin Softmax Loss for Convolutional Neural Networks"**  
*International Conference on Machine Learning (ICML)*  
- **Contribution** : Large margin softmax
- **Avantages** : Meilleure séparation des classes

## 📝 Notes de Recherche

### 1. Concepts Clés

#### Embedding Facial
- **Définition** : Représentation vectorielle du visage
- **Dimension** : Typiquement 128-512 dimensions
- **Distance** : Cosine ou euclidienne

#### Transfer Learning
- **Principe** : Réutilisation modèles pré-entraînés
- **Avantages** : Moins de données nécessaires
- **Fine-tuning** : Adaptation domaine spécifique

#### Data Augmentation
- **Techniques** : Rotation, flip, brightness, noise
- **Purpose** : Améliorer généralisation
- **Impact** : Réduction overfitting
